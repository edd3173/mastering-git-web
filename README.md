# 맞춤식 AI 견적 추천 서비스

## 너무나도 많은 사람들이 견적서 사기를 당하고 있다


실생활에서 많은 사람들은 어떠한 상품을 구매하는데, 우선 컴퓨터 부품을 예로 들고 싶다.(자세한 모델은 차후 변경가능)

컴퓨터 부품을 비롯한 어떤 소비재들의 경우 소비자들이 소위 '덤터기'를 쓰기가 너무나도 쉽다.

따라서 우리의 목적은, 가능한 많은 data를 수집하고 이것을 인공지능에게 학습시킨 뒤에, 소비자가 원하는 맞춤형 견적을 내주는 것이다.

예를 들어, 실제 소비자들은 필요한 performance이상의 부품을 웃돈을 주고 사는 경우가 많다.

----

* RAM : 삼성/하이닉스 부품을 사용하되 RGB가 들어간 번쩍번쩍하거나 이상한 힙한 작은 회사의 부품을 쓸 필요는 없다.
* SSD : 대용량의, 그리고 대기업 제품을 쓰되 PRO 등 과도한 퍼포먼스를 가성비 나쁜 돈을 주고 쓸 필요는 없다.
* CPU : 마찬가지로 가성비가 좋은 것으로 구매하면 된다.
* GPU : 가장 어려운 부분인데, 사용자의 목적에 맞추어서 적당히 사야 하지만, 가격장난과 올려치기를 자주 사용한다.
* Mainboard : 마찬가지로 가성비가 좋은 것으로 구매하면 된다.

----

이러한 상황이 바람직하지만, 실제로 사람들은 실제 상승하는 퍼포먼스에 비해 턱없이 비싼 돈을 주며 비싼 부품을 산다.
설령 가성비 부품이라고 해도 국내의 경우 가격장난이 들어간 경우가 있다. 이 때에는 해외에서 부품을 구매하는 것이 바람직하다.

현재까지의 디자인은 소비자의 컴퓨터 사용목적에 따라 그에 차별화된 맞춤 견적을 내주는 것이다.
